1. Why Quarto? (The Missing Piece)

You now know how to write Python, analyze data, visualize results, and work professionally.

One final question remains:

How do we turn analysis into a polished, shareable product?

This is where Quarto comes in.

2. What Is Quarto?

Quarto is a publishing system for data analysis.

It allows you to:

  • Combine text, code, and results
  • Create reports, dashboards, and websites
  • Render content into HTML
  • Publish work easily to GitHub Pages

Think of Quarto as:

The bridge between analysis and communication.

In this book, we use Quarto with Python.

3. Installing Quarto

Installation Steps

  1. Go to https://quarto.org
  2. Download Quarto for your operating system
  3. Install using default options

Verify Installation

quarto --version

If Quarto responds, installation is successful.

4. How Quarto Documents Work

A Quarto document uses the extension: 

.qmd

It contains three components:

  • YAML — configuration
  • Markdown — narrative text
  • Python code — analysis and visualization

5. YAML Basics (Without Fear)

---
title: "Malaria Dashboard"
author: "Strategic Insights & Analytics (SI Analytics)"
format: html
execute:
  echo: true
---

YAML rules:

  • Always at the top
  • Starts and ends with ---
  • Indentation matters

You do not need to memorize YAML — just recognize patterns.

6. Markdown + Python Integration

Markdown handles text: 

## Introduction
This dashboard explores malaria trends over time.

Python handles analysis: 

import pandas as pd

Quarto runs the code and inserts results automatically.

7. Creating Your First Quarto Dashboard

Project Structure

malaria-dashboard/
├── data/
│   └── malaria.csv
├── dashboard.qmd
└── README.md

YAML Header

---
title: "Malaria Trends Dashboard"
author: "Strategic Insights & Analytics (SI Analytics)"
format: html
execute:
  echo: false
---

Setting echo: false hides code and shows results only.

8. Loading Data in Quarto

import pandas as pd

url = "https://raw.githubusercontent.com/SIAnalyticsLab/30daychallengeguide/main/malaria.csv"
df = pd.read_csv(url)

df.head()

9. Adding Visualizations

import plotly.express as px

fig = px.line(
    df,
    x="year",
    y="cases",
    color="country",
    title="Malaria Cases Over Time"
)

fig

Plotly charts remain fully interactive in HTML output.

10. Structuring a Dashboard

## Overview
This dashboard presents malaria trends by country and year.

- Rising cases in some regions
- Declines in others

## Visual Analysis

Clear structure matters as much as correct analysis.

11. Rendering the Dashboard

quarto render dashboard.qmd

Quarto generates: 

dashboard.html

You now have a professional analytics product.

12. Publishing to GitHub Pages

  2. Push your files
  3. Go to Settings → Pages
  4. Enable Pages from main branch

Your dashboard is now live.
