2. Introduction to Google Colab

2.1 Learning Objectives

  • Understand what Google Colab is and why it’s powerful for data analysis.
  • Learn how to access and navigate the Colab interface.
  • Create and manage notebooks in Google Colab.
  • Run Python code inside code cells.
  • Use text (Markdown) cells for documentation.
  • Import and analyze datasets.
  • Share notebooks with teammates.

2.2 Introduction

Google Colaboratory, or Colab, is a free online platform that lets you run Python code in your browser. There’s no installation required — everything runs on Google’s cloud servers.

Colab is especially powerful for beginners because it removes setup barriers and allows you to practice Python immediately.

While heavy workloads may timeout, Colab is ideal for learning, prototyping, and data analysis workflows.

2.3 Getting Started with Colab

  • Search for “Google Colab” or visit colab.research.google.com.
  • Log in with your Google account.
  • Create a notebook by selecting New Notebook.

2.4 Creating and Managing Notebooks

Notebooks are organized into code cells and text cells. You can rename your notebook from the top-left corner.

2.5 Working with Code Cells

To run code, type in a code cell and click ▶️ or press:

  • Shift + Enter — run and move to next cell
  • Ctrl + Enter — run, stay in same cell

Example Code Cell

# Your first Python command in Colab
print("Hello SI Analytics Bootcamp!")

2.6 Text Cells (Markdown)

Use text cells to add explanations, notes, titles, subtitles, formulas, or documentation. Colab supports Markdown, which is simple and flexible. 

# This is a Markdown heading
## This is a subheading
Here is some *italic*, **bold**, and `inline code`.

2.7 Working with Data

You can upload files from your computer or connect to Google Drive.

Example: Importing a CSV File

import pandas as pd

df = pd.read_csv("Sales_Orders.csv")
df.head()

Use the “Share” button to allow teammates or instructors to view or edit your notebook.

Google Colab is a simple yet powerful environment for running Python for data analysis. It supports code execution, writing documentation, importing data, visualizing outputs, and sharing your work — all from your browser.
