1. Why Functions Exist

Functions allow you to reuse logic instead of repeating code. They make your work cleaner, safer, and easier to scale. 

def greet(name):
    return "Hello " + name

print(greet("Amina"))

2. Functions vs Methods

A function stands alone. A method belongs to an object. 

text = "python"
print(text.upper())

3. Libraries

Libraries extend Python’s power. 

import pandas as pd
import numpy as np
import plotly.express as px
