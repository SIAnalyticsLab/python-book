Up to now, you have used Google Colab.

Colab is excellent for:

  • Learning

However, real projects usually require:

This chapter introduces the tools that professionals use daily.

Learning Python is step one.

Using Python professionally is step two.

2. Installing Python 3.12 (Carefully and Safely)

Why Python 3.12?

  • Latest stable release
  • Performance improvements
  • Better error messages
  • Long-term support

Installation Steps (All Operating Systems)

  1. Go to https://www.python.org/downloads/
  2. Download Python 3.12
  3. During installation, check “Add Python to PATH”
  4. Click Install

Verify Installation

python --version

You should see: 

Python 3.12.x

If you do, installation is successful.

3. What Is Visual Studio Code (VS Code)?

VS Code is a code editor.

It is:

  • Lightweight
  • Powerful
  • Free
  • Widely used in industry

VS Code is not Python itself. It is where you write and manage code.

Why Analysts Use VS Code

  • Python scripts
  • Jupyter notebooks
  • Git integration
  • Extensions
  • Debugging

It is the professional home of your code.

Installing VS Code

Go to https://code.visualstudio.com/, download, and install.

4. Installing the Python Extension

Inside VS Code:

  1. Click the Extensions icon
  2. Search for Python
  3. Install the extension by Microsoft

This enables:

  • Syntax highlighting
  • Code execution
  • Linting
  • Debugging

5. Understanding Project Structure

Professional Python projects are organized. 

project-folder/
│
├── data/
│   └── malaria.csv
│
├── scripts/
│   └── analysis.py
│
├── outputs/
│   └── charts.html
│
├── requirements.txt
└── README.md

This structure:

  • Keeps data separate
  • Makes work reproducible
  • Helps collaborators understand your project

6. Virtual Environments (venv)

What Problem Do Virtual Environments Solve?

Different projects may need different library versions.

A virtual environment isolates dependencies per project.

Think of it as a clean workspace for each project.

Create a Virtual Environment

python -m venv venv

Activate the Environment

Windows

venv\Scripts\activate

macOS / Linux

source venv/bin/activate

Install Packages

pip install pandas plotly numpy

Only this project sees these libraries.

7. Running Python Scripts

Create a file called analysis.py: 

import pandas as pd

print("Professional Python workflow started")

Run it: 

python analysis.py

Notebook vs Script

Notebooks Scripts
Exploration Production
Interactive Automated
Visual Reproducible

Professionals use both—intentionally.

8. Introduction to Git & GitHub

What Is Git?

Git tracks:

  • Changes to files
  • Who changed what
  • When changes happened

Git protects you from lost work and broken code.

What Is GitHub?

GitHub hosts Git repositories, enables collaboration, and publishes content.

You do not need mastery yet—just familiarity.

9. Publishing with GitHub Pages

GitHub Pages allows you to host:

  • HTML files
  • Dashboards
  • Reports
  • Documentation

Basic Steps

  1. Create a GitHub repository
  2. Add an index.html
  3. Go to Settings → Pages
  4. Enable Pages from the main branch

Your content is now public.
